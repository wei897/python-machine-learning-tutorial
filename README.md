# Python机器学习教程

这个项目包含Python机器学习的示例代码，展示了如何使用Python实现各种机器学习算法。

## 文件说明

- `linear_regression_1d.py`: 一维线性回归示例，展示了如何对一维数据进行线性回归分析
- `data_1d.csv`: 用于一维线性回归的数据集

## 如何运行

确保已安装以下依赖：
- Python 3.x
- NumPy
- Matplotlib

运行示例：
```bash
python linear_regression_1d.py
```
